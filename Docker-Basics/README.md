# 🐳 Docker Basics

## What is Docker?

Docker is a platform that allows developers to package applications and their dependencies into lightweight containers.

---

## Why Docker?

- Consistent environments
- Faster deployment
- Lightweight containers
- Easy application sharing
- Better resource utilization

---

## Docker Architecture

- Docker Client
- Docker Daemon
- Docker Images
- Docker Containers
- Docker Registry

---

## Basic Commands

```bash
docker --version
docker info
docker help
docker ps
docker images
```
